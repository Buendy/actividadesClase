PHP 7.0.33 Released Dic, 15, 2018

El equipo de desarrollo de PHP anuncia la disponibilidad inmediata de PHP 7.0.33. Se solucionaron cinco problemas relacionados con la seguridad en esta versión. Se recomienda a todos los usuarios de PHP 7.0 que actualicen a esta versión.

Para las descargas de código fuente de PHP 7.0.33, visite nuestra página de descargas, la fuente de Windows y los archivos binarios se pueden encontrar en windows.php.net/download/. La lista de cambios se registra en el registro de cambios.

Tenga en cuenta que, según la línea de tiempo de soporte de la versión de PHP, PHP 7.0.33 es la última versión programada de la rama de PHP 7.0. Puede haber una versión adicional si descubrimos problemas de seguridad importantes que lo justifican, de lo contrario, esta versión será la última en la rama de PHP 7.0. Si su instalación de PHP se basa en PHP 7.0, puede ser un buen momento para comenzar a hacer los planes para la actualización a PHP 7.1, PHP 7.2 o PHP 7.3.
